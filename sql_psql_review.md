# How to use Postgres

## Launch postgres in MacOS terminal
```psql - U postgres```

<br />

## Common psql commands
|Command | What it does |
|-------|-----------|
|\l | List databases|
|\c name | Select a database |
|\dt | List tables |
|\d table_name | Describe a table |
|\q | Exit psql shell |

<br />
